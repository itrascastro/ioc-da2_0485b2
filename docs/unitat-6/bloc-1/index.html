Introducció a les Estructures de Dades

Les estructures de dades són fonamentals en programació per organitzar, gestionar i emmagatzemar informació de manera eficient. En aquest bloc introductori explorarem els arrays, que són la base de moltes altres estructures de dades més complexes.

Què són les Estructures de Dades?

Una estructura de dades és una forma particular d'organitzar i emmagatzemar dades per facilitar l'accés i les modificacions posteriors. Les estructures de dades defineixen:

  • Com s'organitzen les dades: La disposició física o lògica dels elements
  • Com s'accedeix a les dades: Els mètodes per llegir o modificar elements
  • Quines operacions es poden realitzar: Inserció, eliminació, cerca, ordenació, etc.
  • L'eficiència temporal i espacial: Cost computacional de les operacions

Classificació de les Estructures de Dades

Les estructures de dades es poden classificar en diferents categories:

Tipus Característiques Exemples
Primitives Tipus bàsics del llenguatge int, double, boolean, char
Lineals Elements organitzats seqüencialment Arrays, Llistes, Piles, Cues
No Lineals Relacions jeràrquiques o de xarxa Arbres, Grafs, Taules Hash
Estàtiques Mida fixa en temps de compilació Arrays tradicionals
Dinàmiques Mida variable en temps d'execució ArrayList, LinkedList, HashMap

Els Arrays: Estructures Lineals Estàtiques

Els arrays són la forma més bàsica d'estructura de dades lineal. Consisteixen en una col·lecció d'elements del mateix tipus, emmagatzemats en posicions de memòria consecutives i accessibles mitjançant un índex numèric.

{% include definicio.html terme="Array" definicio="Estructura de dades que conté una col·lecció d'elements del mateix tipus, organitzats seqüencialment i accessibles mitjançant un índex numèric que comença en 0." %}

Avantatges i Inconvenients dels Arrays

Avantatges:

  • Accés directe: Temps constant O(1) per accedir a qualsevol element
  • Eficiència de memòria: Emmagatzematge contigu sense sobrecàrrega
  • Simplicitat: Fàcils d'entendre i implementar
  • Compatibilitat: Suport natiu en gairebé tots els llenguatges

Inconvenients:

  • Mida fixa: No es pot canviar la mida després de la creació
  • Inserció/eliminació costosa: Requereix desplaçar elements O(n)
  • Memòria preasignada: Pot malbaratar espai si no s'usa completament
  • Tipus homogeni: Tots els elements han de ser del mateix tipus

Memòria i Rendiment dels Arrays

Els arrays són extremadament eficients gràcies a la seva organització en memòria:

  • Localitat espacial: Elements consecutius ocupen posicions adjacents de memòria
  • Accés indexat: Fórmula matemàtica: direcció = base + (índex × mida_element)
  • Cache-friendly: Aprofiten la memòria cau del processador per millor rendiment
  • Predicibilitat: El compilador pot optimitzar l'accés als arrays

Algorismes de Cerca

Cerca Lineal (Seqüencial)

La cerca lineal recorre l'array element per element fins trobar el valor buscat:

{% capture code_cerca_lineal %} public static int cercaLineal(int[] array, int valor) { for (int i = 0; i < array.length; i++) { if (array[i] == valor) { return i; // Retorna l'índex on s'ha trobat } } return -1; // No trobat } // Versió que retorna informació detallada public static void cercaLinealDetallada(int[] array, int valor) { int comparacions = 0; boolean trobat = false; int posicio = -1; for (int i = 0; i < array.length; i++) { comparacions++; if (array[i] == valor) { trobat = true; posicio = i; break; } } System.out.println("Valor " + valor + ": " + (trobat ? "trobat a posició " + posicio : "no trobat")); System.out.println("Comparacions realitzades: " + comparacions); } {% endcapture %} {% include code-block.html lang="java" code=code_cerca_lineal %} {% include definicio.html terme="Cerca Lineal" definicio="Algorisme que busca un element recorrent l'array des del primer element fins al últim. Complexitat temporal: O(n)." %}

Cerca Binària

La cerca binària funciona només amb arrays ordenats, dividint repetidament l'espai de cerca per la meitat:

{% capture code_cerca_binaria %} public static int cercaBinaria(int[] array, int valor) { int esquerra = 0; int dreta = array.length - 1; while (esquerra <= dreta) { int mig = (esquerra + dreta) / 2; if (array[mig] == valor) { return mig; // Trobat! } else if (array[mig] < valor) { esquerra = mig + 1; // Buscar a la meitat dreta } else { dreta = mig - 1; // Buscar a la meitat esquerra } } return -1; // No trobat } // Versió recursiva public static int cercaBinariaRecursiva(int[] array, int valor, int esquerra, int dreta) { if (esquerra > dreta) { return -1; // No trobat } int mig = (esquerra + dreta) / 2; if (array[mig] == valor) { return mig; } else if (array[mig] < valor) { return cercaBinariaRecursiva(array, valor, mig + 1, dreta); } else { return cercaBinariaRecursiva(array, valor, esquerra, mig - 1); } } {% endcapture %} {% include code-block.html lang="java" code=code_cerca_binaria %} {% include definicio.html terme="Cerca Binària" definicio="Algorisme eficient per buscar en arrays ordenats que divideix l'espai de cerca per la meitat en cada iteració. Complexitat temporal: O(log n)." %}

Algorismes d'Ordenació

Ordenació Bombolla (Bubble Sort)

Compara elements adjacents i els intercanvia si estan en l'ordre incorrecte:

{% capture code_bombolla %} public static void ordenaBombolla(int[] array) { int n = array.length; boolean intercanviat; for (int i = 0; i < n - 1; i++) { intercanviat = false; // Últims i elements ja estan ordenats for (int j = 0; j < n - 1 - i; j++) { if (array[j] > array[j + 1]) { // Intercanvi int temp = array[j]; array[j] = array[j + 1]; array[j + 1] = temp; intercanviat = true; } } // Si no hi ha hagut intercanvis, l'array ja està ordenat if (!intercanviat) { break; } } } // Versió amb informació de debug public static void ordenaBombollaDetallada(int[] array) { int n = array.length; int passades = 0; int intercanvis = 0; for (int i = 0; i < n - 1; i++) { passades++; boolean intercanviat = false; System.out.println("Passada " + passades + ": " + java.util.Arrays.toString(array)); for (int j = 0; j < n - 1 - i; j++) { if (array[j] > array[j + 1]) { int temp = array[j]; array[j] = array[j + 1]; array[j + 1] = temp; intercanviat = true; intercanvis++; } } if (!intercanviat) break; } System.out.println("Resultat final: " + java.util.Arrays.toString(array)); System.out.println("Passades: " + passades + ", Intercanvis: " + intercanvis); } {% endcapture %} {% include code-block.html lang="java" code=code_bombolla %}

Ordenació per Selecció (Selection Sort)

Troba repetidament l'element mínim i l'ubica al principi:

{% capture code_seleccio %} public static void ordenaSeleccio(int[] array) { int n = array.length; for (int i = 0; i < n - 1; i++) { // Trobar l'índex del mínim element int indexMin = i; for (int j = i + 1; j < n; j++) { if (array[j] < array[indexMin]) { indexMin = j; } } // Intercanviar l'element mínim amb el primer element if (indexMin != i) { int temp = array[i]; array[i] = array[indexMin]; array[indexMin] = temp; } } } {% endcapture %} {% include code-block.html lang="java" code=code_seleccio %}

Ordenació per Inserció (Insertion Sort)

Construeix l'array ordenat element per element, inserint cada nou element en la seva posició correcta:

{% capture code_insercio %} public static void ordenaInsercio(int[] array) { for (int i = 1; i < array.length; i++) { int clau = array[i]; int j = i - 1; // Moure elements majors que 'clau' una posició endavant while (j >= 0 && array[j] > clau) { array[j + 1] = array[j]; j--; } array[j + 1] = clau; } } // Versió detallada que mostra el procés public static void ordenaInsercioDetallada(int[] array) { System.out.println("Array inicial: " + java.util.Arrays.toString(array)); for (int i = 1; i < array.length; i++) { int clau = array[i]; int j = i - 1; System.out.println("Inserint " + clau + " en posició correcta..."); while (j >= 0 && array[j] > clau) { array[j + 1] = array[j]; j--; } array[j + 1] = clau; System.out.println("Després del pas " + i + ": " + java.util.Arrays.toString(array)); } } {% endcapture %} {% include code-block.html lang="java" code=code_insercio %}
